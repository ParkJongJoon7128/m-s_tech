# BLE 통신 프로그램
공기청정기와 자동차 필터에 부착한 단말기와 어플리케이션의 BLE 통신으로, 단말기와 어플리케이션이 BLE 통신에 성공하면 모바일에서 접속한 Wifi SSID와 PWD를 단말기로 보내 접속한 WIFI로 단말기가 연결할수 있도록 커넥터 역할을 해주는 (주)앰엔에스테크에서 개발한 어플리케이션입니다.

# 사용기술
- Android
- BLE
